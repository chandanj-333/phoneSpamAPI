Steps to Run:

1.Unzip the folder
2.make sure django and djangorestframework is installed on machine
3.move to InstahyreTask_Chandan/PhoneSpam
4.make the migrations (python manage.py makemigrations)
5.migrate the changes (python manage.py migrate)
6.run the development server (python manage.py runserver) and make sure the server is running
7.check the urls.py under PhoneSpam folder for different endpoints and their uses
	this contains both endpoints and corresponding classes associated with it
8. check views.py under PhoneSpam folder to check the payload and methods supported for endpoints
9.can test the API's using any API testing tool(postman,insomia etc)
